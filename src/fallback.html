Drupal Ecosystem

Drupal is more than a CMS — it is a framework, a global community of initiatives and local organizations, and an ecosystem of companies that build products and services around it.

Code

Drupal Core, contributed modules, and the tools developers use to build and run Drupal sites.

Drupal Core

The core software: PHP framework, content modeling, rendering, configuration, and the foundational modules every Drupal site uses.

Contributed modules

Tens of thousands of community-maintained modules and themes that extend Drupal — from Views and Paragraphs to commerce, search, and AI integrations.

Tools

Command-line and local-development tools the Drupal community builds and shares.

Drush

The Drupal Shell — the standard command-line tool for administering Drupal sites: cache clears, config sync, user management, scripting.

DDEV

A Docker-based local development environment widely used by Drupal teams; one of several local-dev tools the community relies on.

Community

The people, initiatives, and organizations behind Drupal — globally and regionally.

Initiatives

Project-wide efforts coordinated by community members and the Drupal Association.

Composer Support in Core

Initiative to make Composer the supported way to install and update Drupal Core.

Drupal AI

A growing set of contributed modules and an associated initiative bringing AI capabilities into Drupal site building and editorial workflows.

Innovation eXchange Program (IXP)

Community program coordinating innovation work and contributions across the project.

Community Working Group

Volunteer group that supports the health of the Drupal community and upholds its code of conduct.
