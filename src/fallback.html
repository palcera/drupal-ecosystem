Drupal Ecosystem

Drupal is more than a CMS — it is a framework, a global community of initiatives and local organizations, and an ecosystem of companies that build products and services around it.

Framework

Drupal Core, contributed modules, and the tools developers use to build and run Drupal sites.

The core software: PHP framework, content modeling, rendering, configuration, and the foundational modules every Drupal site uses.

Dev Tools

Drush

The Drupal Shell — the standard command-line tool for administering Drupal sites: cache clears, config sync, user management, scripting.

Documentation

Placeholder — documentation tooling and resources (api.drupal.org, the Drupal User Guide, contributed-module docs).

Composer

PHP dependency manager — the supported way to install, update, and patch Drupal Core and contributed projects.

Lando

Another Docker-based local-dev environment popular with Drupal teams, particularly those coming from Pantheon-style workflows.

Drupal Console

Symfony-Console-based scaffolding CLI. Widely used during Drupal 8/9 era; largely superseded by Drush + Composer + Recipes today.

Recipes

Drupal 10.3+ mechanism for installable, repeatable site configurations — modules, config, and content together. Powers Drupal CMS.

api.drupal.org

Generated API documentation site for Drupal Core and major contributed modules — the developer reference.

Drupal CMS

Drupal's official 2025 distribution targeting marketers, designers, and site builders with smart defaults, recipes, and a guided setup. The fastest way for a non-developer to stand up a Drupal site.

Drupal AI

Drupal's AI product line — contributed modules and an associated initiative bringing AI capabilities (assistants, content generation, agents) into Drupal site building and editorial workflows.

Drupal Commerce

Mature e-commerce framework for Drupal, maintained by Centarro. Powers everything from boutique stores to enterprise commerce on top of Drupal's content modeling.
