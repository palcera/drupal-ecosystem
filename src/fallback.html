Drupal Ecosystem

Drupal is more than a CMS — it is a framework, a global community of initiatives and local organizations, and an ecosystem of companies that build products and services around it.

Code

Drupal Core, contributed modules, and the tools developers use to build and run Drupal sites.

Drupal Core

The core software: PHP framework, content modeling, rendering, configuration, and the foundational modules every Drupal site uses.

Action

Trigger configurable actions on events (publish, email, etc.).

Block

Place reusable content blocks into theme regions.

Book

Build hierarchical, multi-page documentation structures.

Breakpoint

Manage responsive-design breakpoints used by themes and media.

CKEditor 5

Rich-text WYSIWYG editor integration (current default).

Configuration

Manage site configuration via importable/exportable YAML.

Contact

Site-wide and per-user contact forms.

Contextual Links

Database Logging

Editor

Field

File

Help

History

Image

JSON:API

Layout Builder

Link

Media
